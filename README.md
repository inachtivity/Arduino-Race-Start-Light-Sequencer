This project simulates a motorsport-style race start light sequence using an Arduino. 

## Features:
- Sequential LED countdown (5 red lights)
- Randomized delay before lights out (200 - 3000 ms)
- Push button to trigger the start of the sequence

## Hardware Requirements:
- Arduino Board
- 5 red LED Lights
- 5 resistors (220Ω)
- Push button
- Breadboard
- Jumper wires
